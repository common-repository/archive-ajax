Adds a new AJAXified widget for archive.

== Description ==

This plugin adds a Widget for an AJAXified version of archive.

== Installation ==

Opt 1:

1. Go to Plugins -> Menu -> Add new
2. Search for `archive ajax`
3. Install the plugin
4. Activate the plugin

Opt 2:

1. Download the plugin from http://downloads.wordpress.org/plugin/archive-ajax.zip
2. Upload `archive-ajax` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
